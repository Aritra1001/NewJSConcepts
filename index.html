What is Function Currying?

Function Currying is a javascript mechanism in which a function having multiple arguments is transformed into a series of nested functions each having a single argument.

Normally we write:

        function addition(a,b,c){
            return a + b + c;
        }
        let res = addition(2,3,4);
        console.log(res);

Using function currying

        function addition(a){
            return function(b){
                return function(c){
                    return a + b + c;
                }
            }
        }
        
        let res = addition(2)(3)(4);
        console.log(res);

What is function generator?

A generator function is used to generate value and it returns a generator object. The yield keyword used within the generator function is used to pause and resume the execution of code. 

        function* simpleGenerator(){
            console.log("generator function");
            yield 20;
            let x = "second step"
            yield x;
            yield 40;
        }
        let sg = simpleGenerator();
        console.log(sg.next());
        console.log(sg.next());
        console.log(sg.next());
        console.log(sg.next());

What is nullish coalescing operator?

This operator is used to check only undefined or null value. It won't check boolean or empty string. The nullish-coalescing operator is a logical operator that return its right hand side operand when its left side operand is either null or undefined, otherwise returns its left hand side operand. 

        let user={
            name: "",
            age: 0
        }
        console.log(user.name ?? "unknown"); 
        console.log(user.age ?? 21);

Bubbling is a concept of DOM( Document Object Model). In bubbling, the event is first captured by the innermost element and then propagated to outer elements.

What is local storage and session storage?

Local storage - Local storage is the property in javascript that allows javascript site or apps to save key-value pairs on the browser without any expiration date. The data can be retrieved even after the user closes the browser or restarts the computer.

Session storage - Session storage is a popular choice for storing data in the browser. The data will be stored for a particular session only. The data is cleared once the user closes the browser window.
